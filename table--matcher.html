Important - How to Run This Application:

This application has a local server (`server.py`) and this frontend. The "Preview" button will not work because it cannot connect to your local server.

  1. Save both `server.py` and this HTML file in the same folder.
  2. Run the server from your terminal: `python server.py`
  3. Open your web browser and go to this exact address: http://127.0.0.1:5001
